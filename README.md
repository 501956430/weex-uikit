## Dev steps
1. clone
```shell
git clone git@github.com:erguotou520/weex-uikit.git
```
2. install dependencies
```shell
cd weex-uikit
npm i
```
3. run demo
```shell
npm run demo:dev
```
3. debug
```shell
npm run demo:debug
```
